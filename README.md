### Icons

1. Open https://icones.js.org/collection/ic find icon you need
   
2. Install icon set
   
   ```bash
   yarn add -D @iconify-json/ic
   ```

3. Use in template
   
   ```html
   <span i-ic-baseline-add-circle text-gray-500></span>
   ```
